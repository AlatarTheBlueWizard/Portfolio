Mobile UX notes

  • Costly and time consuming for a full rebuild
  • Provides best experience across multiple devices
  • Addresses constraints of mobile size bandwith
  • Future friendly and ready for growth
  • Responsive != mobile first!
  • Start small and scale up
  • Be there, Be useful, Be Quick!
  • 75% of people rely on thumbs, 49% use one handed grip
  • Reduce image usage throughout sites/apps
  • SVG/Webfonts are lower bandwith
  • Optimize and Minify CSS and JS files
  • GZIP files to eliminate unecessary data
  • Make sure most commonly used menu items are located at the bottom
  • When taking numbers, ensure the number keypad is used on mobile
  • Default keyboard make it cumbersome to enter numbers only
  • Stop using Pop-ups/Pop-overs
  • Users don't visit your website to look at ads
  • Keep it simple and sleek
  • Make sure it's unobtrusive. Content first
